# v0.5.21 — Equipe cadastrada em formato de lista

- A área Equipe cadastrada voltou a usar uma lista vertical.
- Cada profissional ocupa uma linha completa.
- As categorias continuam separadas por cabeçalhos compactos.
- A lista mantém rolagem interna para não alongar a página.
- Ao abrir um profissional, os detalhes aparecem abaixo da própria linha.
- Nenhuma informação, filtro ou ação administrativa foi removida.
