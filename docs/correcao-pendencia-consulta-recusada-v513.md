# Consulta recusada nas Pendências — v0.5.13

- Solicitações com status de recusa passam a ser classificadas como pendência no Portal do Paciente.
- A resposta administrativa gravada no payload da consulta é retornada pela API do paciente.
- O motivo/orientação da recusa é exibido no detalhamento da pendência.
- Consultas recusadas deixam de aparecer indevidamente em Consultas marcadas.
- Nenhuma tabela, coluna ou política foi criada ou alterada.
